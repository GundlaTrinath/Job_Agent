## Getting Started

### First Time Setup

1. **Open the Application**
   - Visit the deployed URL or `http://localhost:5173` for local

2. **Create Your Profile**
   - Click "Profile" in the navigation
   - Fill in your details:
     - Name and email
     - Current role
     - Location
     - Skills
     - Salary expectations

3. **Upload Your Resume** (Optional)
   - Go to "Resume" page
   - Upload PDF resume
   - Get instant AI feedback

4. **Start Chatting!**
   - Navigate to Chat (home icon)
   - Ask the AI to find jobs for you

## Job Board

View and manage all discovered job opportunities.

### Features

#### Job Cards Display
Each job shows:
- **Job Title & Company**
- **Location**
- **Salary Range** (in ‚Çπ Rupees)
- **Description**
- **Required Skills** (tags)
- **Application Status**

#### External Job Indicator
Look for the "External Link" badge - these are real jobs from company websites!

### How to Apply

#### For Web Jobs (with External Link)
1. Click **"Apply Now"**
2. Opens company career page in new tab
3. Fill application on their website
4. Get real interview calls!

#### For Internal Jobs
1. Click **"Easy Apply"**
2. Application status saved
3. Track in dashboard

### Job Statuses
- **Active**: Not yet applied
- **Applied**: Application submitted

### Empty State
If you see "No jobs yet":
- Click **"Start Chat"**
- Ask AI to find jobs for you

## Skill Tests

Assess and improve your skills with AI-generated tests.

### Test Library

Each test shows:
- **Skill Name** (e.g., "Python Assessment")
- **Number of Questions**
- **Estimated Time**
- **Difficulty Level**
- **Last Score** (if taken before)

### Taking a Test

1. **Click "Take Test"**
2. **Answer Questions**
   - Multiple choice format
   - Progress bar shows completion
   - Timer tracks duration
3. **Submit When Complete**
   - All questions must be answered
4. **Review Results**
   - See score and percentage
   - Detailed feedback per question
   - Correct answers shown
   - Explanations provided

### Test Results

- **Pass**: 60% or higher (green badge)
- **Below Pass**: Under 60% (orange/red badge)

### Retry Tests
You can retake any test to improve your score!

## Tips & Best Practices

### Maximize Job Search Results

1. **Complete Your Profile First**
   - More accurate matches
   - Better salary recommendations

2. **Be Specific in Queries**
   - ‚ùå "find jobs"
   - ‚úÖ "find senior React developer jobs in Bangalore with 5+ years experience"

3. **Check Job Board Regularly**
   - New jobs added as you search
   - Apply quickly for best chances

### Level Up Your Skills

1. **Take Tests Before Applying**
   - Know your current level
   - Identify weak areas
   - Practice before interviews

2. **Follow Learning Paths**
   - Structured roadmap
   - Curated resources
   - Track progress

3. **Practice Interview Questions**
   - Ask AI for common questions
   - Take skill tests multiple times

### Improve Resume Score

1. **Upload and Review Regularly**
   - Update after new projects
   - Refine based on feedback

2. **Tailor to Job Descriptions**
   - Match keywords from job posts
   - Highlight relevant experience

3. **Quantify Everything**
   - Use numbers and percentages
   - Show impact, not just duties

### Make the Most of AI Chat

1. **Ask Open-Ended Questions**
   - AI provides detailed responses
   - Learn more about career paths

2. **Request Explanations**
   - Don't understand something? Ask!
   - AI explains concepts simply

3. **Multi-Step Conversations**
   - Build on previous messages
   - AI remembers context

## Troubleshooting

### Jobs Not Loading
- Check internet connection
- Refresh the page
- Try a different search query

### Resume Upload Failed
- Ensure file is PDF
- Check file size (max 10MB)
- Try a different browser

### Test Won't Submit
- Answer all questions
- Check for error messages
- Refresh if stuck

### AI Not Responding
- Check backend server is running
- Verify API key is set
- Look for error in chat
